# bc1beat.com

Personal website showcasing work in the agentic economy.

## Stack

- Pure HTML/CSS
- No JavaScript
- No frameworks

## Local Development

```bash
# Open in browser
open index.html

# Or use any static server
python3 -m http.server 8000
```

## Deploy

Push to GitHub and enable GitHub Pages, or deploy to any static host (Vercel, Netlify, Cloudflare Pages).

## Structure

```
/
├── index.html    # Single page site
├── styles.css    # Terminal-style dark theme
└── README.md
```
