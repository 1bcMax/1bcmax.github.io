Building the payment layer for the agentic economy

GitHub X / Twitter BlockRun

Agents need to pay for services autonomously. Traditional payment systems require account creation, credit cards, and KYC verification — none of which AI agents can complete.

The x402 protocol (HTTP 402 "Payment Required") enables cryptographic micropayments embedded directly in HTTP requests. Pay and get response in a single interaction. No accounts. No API keys. Just wallets and signatures.

This is the missing infrastructure for the agentic economy — where autonomous systems manage their own computational resources and expenses at scale.

// The End State

We are building the economic layer for autonomous agents.

For agents to operate without human intervention, they require identity, payments, and service discovery. BlockRun provides the infrastructure for the latter two:

  • ClawRouter — Our service (LLM routing) which addresses immediate cost concerns
  • x402 — A payment primitive applicable to any API, not just LLMs
  • Discovery Layer — Enables agents to find services by capability rather than name

While ClawRouter is our entry point due to LLM costs being a primary pain point, these rails support any API an agent requires — including data feeds, compute, and storage.

Our goal is for x402 to become the standard for agent commerce.
