// Thesis

Agents need to pay for services autonomously. Traditional payment systems require account creation, credit cards, and KYC verification — none of which AI agents can complete.

The x402 protocol (HTTP 402 "Payment Required") enables cryptographic micropayments embedded directly in HTTP requests. Pay and get response in a single interaction. No accounts. No API keys. Just wallets and signatures.

This is the missing infrastructure for the agentic economy — where autonomous systems manage their own computational resources and expenses at scale.
